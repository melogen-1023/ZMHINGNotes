#### 二叉树的性质

**非空二叉树叶子结点数等于度为2的结点数加1，即N0 = N2 + 1**

非空二叉树上第K层上至多有2^(k-1)个结点(即考虑满二叉树)

高度为h的二叉树至多有2^(h)-1个结点(即考虑满二叉树)

### 二叉树的遍历

#### 先序遍历

访问顺序为根结点, 左子树，右子树

#### 中序遍历

访问顺序为左子树，根结点，右子树
